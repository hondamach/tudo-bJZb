Clone the repo: git clone https://github.com/hondamach/tudo.git

Go into the directory: cd tudo

Execute: ./RUNME.sh

Look in the terminal output and you should see an IP address: AH00558: apache2: ... 172.17.0.2 ...' That is the target!
